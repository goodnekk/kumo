ABOUT THE LANGUAGE

Kumo is a language designed to make things happen. Verbs are best for expressing actions, that is why in kumo allmost everything is a function.

For example, all math is done using functions. If you prefer you can use simple infix notation but this is converted during compilation. These two lines are equivalent:

a = +(55,45)
a = 55+45

This makes operator overloading and extention very simple. Just define the function using an operator:

^ = (a,b){
  repeat(b-1,{
    a = a * a
  })
  return(a)
}

square = 5^2


Functions can be passed around as arguments:

a = 10

subtract = {
  a=a-1
  say(a)
}

while(a>0, subtract)
